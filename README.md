# open-data-ai-analytics

## Мета
Використання системи контролю версій Git. Проєкт з завантаження, перевірки якості, дослідження та візуалізації відкритих даних з джерела data.gov.ua.

## Джерело даних
Дані з системи “Електронна черга перетину кордону” (єЧерга) — data.gov.ua  
(ресурси: queue.csv, busQueue.csv)

## Питання / гіпотези для аналізу
1) У які години/дні спостерігаються пікові навантаження в черзі?
2) Які пункти пропуску мають найбільші середні/медіанні значення часу очікування (або найбільшу інтенсивність подій)?
3) Чи відрізняється динаміка/навантаження між типами транспортних засобів (наприклад, автобуси vs загальна черга)?
